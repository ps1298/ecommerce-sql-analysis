# 🛒 E-Commerce Sales Analysis (PostgreSQL)

## 📊 Overview

This project analyzes e-commerce sales data using SQL to uncover key business insights related to customers, products, and revenue.

## 🎯 Objectives

* Analyze customer purchasing behavior
* Identify top-performing products and categories
* Track revenue trends over time
* Apply advanced SQL techniques

## 🛠️ Skills Used

* Joins (INNER JOIN)
* Aggregations (SUM, COUNT)
* GROUP BY & HAVING
* Common Table Expressions (CTE)
* Window Functions (RANK, PARTITION BY)
* Date functions (DATE_TRUNC)

## 🔍 Key Business Questions

* Who are the top customers by revenue?
* What are the best-selling products?
* Which category generates the most revenue?
* How does revenue change monthly?

## 📈 Key Insights

* The top customer generated $1800 in total revenue
* Electronics category contributes the highest revenue
* A small number of products drive most sales
* Revenue shows a monthly trend pattern
